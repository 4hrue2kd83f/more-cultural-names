This is a mod for various strategy games, that adds new place names based on their culture/faction.

The names are meant to provide an immersive and accurate experience for the game's setting and time period.

There are currently [b]over 60 thousand[/b] names for [b]over 450[/b] languages, settings and time periods, in the main common database.

[img=https://raw.githubusercontent.com/hmlendea/more-cultural-names/master/assets/banner_compatibility.png][/img]

The mod is [b]OS-agnostic[/b], which means that it will work on all of the operating systems supported by the game.

It is actively tested (and developed) on [b]Linux[/b], where it is guaranteed to work with either the native version of the game or through Steam Play.

It is also [b]save-game compatible[/b], and can be added and removed at will without breaking anything.
