[h1]Description[/h1]

This is a mod for various strategy games, that adds new province names based on their culture/faction.

The names are meant to provide an immersive and accurate experience for the game's time period

Supported games:
 - [url=https://steamcommunity.com/sharedfiles/filedetails/?id=1175098675]Crusader Kings 2 with ([b]required[/b]) Historical Immersion Project mod[/url]
 - [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2217534250]Crusader Kings 3[/url]
 - [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2219177532]Imperator: Rome[/url]


The common database currently has over [b]26 thousand[/b] names for over [b]300[/b] languages and time periods.

[h1]Manual installation[/h1]

You can download the latest release from the [url=https://github.com/hmlendea/more-cultural-names/releases]GitHub page[/url] and extract the contents into the appropriate mod directory for your game.

Note: This mod gets updated more frequently on GitHub than on the Steam Workshop pages

[h1]Compatibility[/h1]

The mods are OS-agnostic, which means that they will work on all of the operating systems supported by the game.

The mod is also actively tested (and developed) on Linux, where it will work with the native versions of the games, or through Steam Play if needed.
